# เกมส์ทายตัวเลข (Number Guessing Game)

## ความเป็นมา

โปรแกรมเกมส์ทายตัวเลข 1-100 เขียนด้วย HTML, CSS และ Vanilla JavaScript

## เวอร์ชัน

- v1: เวอร์ชันพื้นฐาน
- v2: เพิ่ม Bootstrap, validation, counter

## วิธีการใช้

1. เปิด `index.html` ในเบราว์เซอร์
2. พิมพ์ตัวเลข แล้วคลิก "ทาย"
3. โปรแกรมจะบอกว่าสูงไป/ตํ่าไป/ถูกต้อง

## ข้อกําหนด

- HTML5
- CSS3 (Bootstrap 5)
- JavaScript ES6+
